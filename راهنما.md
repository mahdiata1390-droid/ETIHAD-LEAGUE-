# راهنمای راه‌اندازی — لیگ من

سلام رفیق 👋 این پروژه یه اپ وب واقعیه که آماده‌ست بذاریش آنلاین و بعد ازش APK بگیری. فقط چند مرحله مونده که همش رایگانه.

## 📱 نسخه گوشی (بدون کامپیوتر)

کل این کار با گوشی هم قابل انجامه، فقط به جای Netlify از **GitHub** استفاده می‌کنیم چون ویرایش فایل و آپلود روش راحت‌تره:

1. فایل zip رو با اپ **Files by Google** (یا هر مدیر فایل دیگه) باز و **Extract/استخراج** کن تا پوشه‌ش دربیاد.
2. برو [github.com](https://github.com) با مرورگر گوشیت (کروم بهتره)، یه اکانت رایگان بساز.
3. بالا سمت راست، دکمه‌ی `+` رو بزن → **New repository**. یه اسم بذار (مثلاً `league-app`) → **Create repository**.
4. تو صفحه‌ی ریپو، **Add file > Upload files** رو بزن و همه‌ی فایل‌های داخل پوشه‌ای که استخراج کردی (index.html, manifest.json, sw.js, آیکون‌ها) رو انتخاب و آپلود کن → پایین صفحه **Commit changes** بزن.
5. برای ویرایش کلیدهای Firebase داخل `index.html`: تو همون ریپو روی فایل `index.html` بزن، بعد آیکون مداد (✏️) بالا سمت راست رو بزن، مقادیر `PASTE_YOUR_...` رو با کلیدهای واقعی Firebase عوض کن (مرحله ۲ پایین رو ببین)، و **Commit changes** بزن.
6. برای آنلاین شدن: برو **Settings > Pages** (تو منوی سمت چپ ریپو). زیر "Branch" گزینه `main` و پوشه `/ (root)` رو انتخاب کن → **Save**. بعد چند دقیقه صبر کن، یه لینک بهت میده شبیه `https://username.github.io/league-app/`.
7. همون لینک رو بده به PWABuilder (مرحله ۶ پایین) تا APK بگیری.

همه این مراحل کاملاً تو مرورگر گوشی انجام میشه، نیازی به هیچ اپ خاصی نیست.

---

## مرحله ۱: ساخت پروژه Firebase (دیتابیس آنلاین رایگان)

1. برو به [console.firebase.google.com](https://console.firebase.google.com) و با اکانت گوگلت وارد شو.
2. روی **Add project** بزن، یه اسم بذار (مثلاً `league-app`) و پروژه رو بساز (نیازی به فعال کردن Google Analytics نیست، می‌تونی خاموشش کنی).
3. تو پنل پروژه، از منوی سمت چپ برو روی **Build > Firestore Database**.
4. روی **Create database** بزن، حالت **Production mode** رو انتخاب کن (بعداً قانونش رو عوض می‌کنیم)، و یه location نزدیک به خودت انتخاب کن.
5. بعد از ساخته شدن دیتابیس، برو تب **Rules** و این رو جایگزین متن موجود کن، بعد **Publish** بزن:

```
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /leagues/{leagueId} {
      allow read: if true;
      allow write: if true;
    }
  }
}
```

> این قانون یعنی هرکی لینک اپ رو داشته باشه می‌تونه بخونه و بنویسه — رمز PIN توی خود اپ جلوی ویرایش رو می‌گیره، نه این قانون. برای یه لیگ دوستانه کافیه، ولی امنیت واقعیِ سطح بانکی نیست.

## مرحله ۲: گرفتن کلیدهای پروژه

1. تو کنسول Firebase، روی چرخ‌دنده ⚙️ کنار "Project Overview" بزن و برو **Project settings**.
2. پایین صفحه، بخش **Your apps**، روی آیکون `</>` (Web) بزن.
3. یه نام بذار (مثلاً `league-web`) و **Register app** بزن.
4. یه بلوک کد بهت نشون می‌ده شبیه این:

```js
const firebaseConfig = {
  apiKey: "AIza...",
  authDomain: "league-app-xxxx.firebaseapp.com",
  projectId: "league-app-xxxx",
  storageBucket: "league-app-xxxx.appspot.com",
  messagingSenderId: "...",
  appId: "..."
};
```

5. این مقادیر رو کپی کن.

## مرحله ۳: چسبوندن کلیدها تو پروژه

فایل `index.html` رو باز کن (با هر ادیتور متنی، حتی Notepad)، دنبال این بخش بگرد:

```js
window.FIREBASE_CONFIG = {
  apiKey: "PASTE_YOUR_API_KEY",
  ...
};
```

و مقادیری که از Firebase گرفتی رو جایگزین کن. ذخیره کن.

## مرحله ۴: آنلاین کردن سایت (هاست رایگان)

ساده‌ترین راه، **Netlify Drop** هست، بدون نیاز به اکانت یا نصب چیزی:

1. برو به [app.netlify.com/drop](https://app.netlify.com/drop)
2. کل پوشه‌ی این پروژه (فایل‌های index.html، manifest.json، sw.js، آیکون‌ها) رو بکش و بنداز تو صفحه.
3. چند ثانیه بعد یه لینک بهت می‌ده مثل `https://random-name-123.netlify.app` — این لینک واقعیِ سایتته.

(اگه راحت‌تری، GitHub Pages هم گزینه‌ست، ولی Netlify Drop سریع‌تره.)

## مرحله ۵: تست کردن

لینک رو باز کن، اولین بار که بازش می‌کنی ازت یه رمز مدیریت می‌خواد — همون رمزیه که فقط خودت باید بدونی. بعدش تیم و بازی اضافه کن و ببین رو گوشی دیگه‌ای هم (بدون وارد کردن رمز) جدول زنده به‌روز میشه یا نه.

## مرحله ۶: گرفتن APK

1. برو به [pwabuilder.com](https://www.pwabuilder.com)
2. لینک سایتت (همون لینک Netlify) رو وارد کن و Enter بزن.
3. چند ثانیه صبر کن تا امتیاز PWA رو نشون بده.
4. روی **Package for stores** بزن و **Android** رو انتخاب کن.
5. تنظیمات پیش‌فرض معمولاً خوبن؛ در آخر یه فایل APK دانلود می‌کنی که می‌تونی مستقیم رو گوشی نصبش کنی (یا بفرستیش برای بقیه).

## نکته آخر

اگه گوشیت اجازه نصب از "منابع ناشناس" رو نمی‌ده، تو تنظیمات گوشی باید موقتاً فعالش کنی تا بتونی APK رو نصب کنی.

هر جا گیر کردی بگو، باهم حلش می‌کنیم 🔥
